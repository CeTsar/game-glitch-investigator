# Game Glitch Investigator

## Project Overview

This project is a beginner Python debugging project. The goal was to investigate a game that had glitches, fix the problems, and add automated tests.

I made a number guessing game where the player guesses a number and gets hints. The game uses separate functions for checking guesses and updating the score so the logic is easier to test.

## Demo Walkthrough

To run the game:

```bash
python game.py
```
